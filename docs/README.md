# 기능 목록

- [x] 쉼표(,)를 기준으로 구분
- [x] 사다리 만들기
  - [x] 라인이 겹치지 않도록
  - [x] 랜덤으로 라인 생성
- [x] 사다리 결과 판단
  - [x] Line에서 이동

## 입력

- [x] 참여하는 사람의 이름
- [x] 사다리 높이
- [x] 실행 결과 입력
- [x] 결과를 보고 싶은 사람

## 출력

- [x] 사다리 결과
- [x] 게임의 결과
  - [x] 개인
  - [x] 전체(all)

## 예외

- [x] 참여하는 사람의 이름이 5자를 초과한 경우
- [x] 참여하는 사람의 이름이 비어있는 경우
- [x] 실행 결과의 개수가 참여한 인원 수와 일치하지 않는 경우
